Driven
=====

|Documentation Status||Build Status||Coverage Status|

**Driven** is a high-level python library developed to integrate and
analyze *omics* data using constraint-based modeling. The library provides
a state-of-the-art simulation methods, access to models and workflows to
easy analyze the most common data-types.

Integration with **escher** allows easy visualization of results, comparison
between different conditions or methods.
